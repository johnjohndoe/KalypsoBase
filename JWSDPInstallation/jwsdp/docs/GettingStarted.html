Getting Started with JavaTM Web Services Developer Pack 1.4

Introduction

The JavaTM Web Services Developer Pack (Java WSDP) is a single download that contains the following key technologies you can use to simplify building of Web services using the Java 2 Platform:

  • XML Digital Signature Version 1.0 EA
  • XML and Web Services Security Version 1.0 EA3
  • WS-I Attachments Sample Application Version 1.0 EA
  • JAXB Version 1.0.3 (Java Architecture for XML Binding)
  • JAXP Version 1.2.6 (Java API for XML Processing)
  • JAXR Version 1.0.6 (Java API for XML Registries)
  • JAX-RPC Version 1.1.2 (Java API for XML-based RPC)
  • SAAJ Version 1.2.1 (SOAP with Attachments API for Java)
  • JSTL Version 1.1 (JavaServer PagesTM Standard Tag Library)
  • Java WSDP Registry Server Version 1.0_06

Java WSDP v1.4 runs in the following Web containers:

  • Sun Java System Application Server Platform Edition 8
  • Sun Java System Web Server 6.1
  • Tomcat 5.0 for Java WSDP
  • No Web container

Select the container into which you integrated the Java WSDP 1.4 component technologies to see how to quickly deploy your own Web applications, try out tools that come with the Java WSDP, and see where to get the documentation that more completely explains how to use the technologies in the Java WSDP.
