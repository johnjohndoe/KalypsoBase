Getting Started with JavaTM Web Services Developer Pack 1.4 on Sun Java System Web Server 6.1

Introduction

You can use this page to quickly deploy your own Web applications, start the Sun Java Systems Web Server 6.1, and see where to get the documentation that more completely explains how to use the technologies in the Java WSDP. The instructions on this page assume that you have already performed a "Typical" installation (installed all of the components).

Deploying Your Own Web Applications

See "Running Java Web Applications On Your Server" to see how to deploy and run Web applications on the Sun Java System Web Server 6.1.

Starting the Web Server

Starting the Web Server entails starting the Administration Server and then starting the Web Server. See "Start the Administration Server" and "Start the Web Server Instance".

Once the Web Server is up and running, point your browser to http://localhost:8080/JWSDP.html. The page that is displayed contains links to several sample programs.

Administration Tools

See the Sun Java System Web Server 6.1 Administration Guide for detailed information about using the Administration Server, the Web Server's main administration tool.

Accessing the Java WSDP Documentation

See the index page at <JWSDP_HOME>/docs/index.html for links to the API documentation, release notes, and reference documentation for the Java WSDP technologies.

Visit this website for news about soon to be released technologies, such as WebRowSet, that you can use with Java WSDP v1.4.

Please send feedback on the Java Web Services Developer Pack to users@jwsdp.dev.java.net.
