JSR 105 v0.12 API Changes

javax.xml.crypto.dsig

  1. Added new XMLSignature.SignatureValue nested class that represents the SignatureValue element and has methods for accessing the value and the Id attribute, as well as a validate method for validating the signature value.
  2. Changed the return type of the XMLSignature.getSignatureValue method from byte[] to SignatureValue.
  3. Removed the XMLSignature.getSignatureValueId method (use new SignatureValue.getId method instead).
  4. Renamed the getCalculatedSignatureValue method of the SignedInfo class to getCanonicalizedData to reflect its true purpose.
  5. Moved the getKeySelectorResult method from the SignedInfo class to the XMLSignature class.
  6. Moved the validate method from the SignedInfo class to the new SignatureValue class.

javax.xml.crypto.dsig.dom

  1. Added package description.

javax.xml.crypto.dsig.keyinfo

  1. Changed the KeyInfoFactory.newKeyValue method to throw KeyException if the key algorithm is unrecognized or unsupported.
  2. Clarified that the KeyInfo.getContent method returns instances of public XMLStructure subclasses.
  3. Changed the RetrievalMethod.dereference method to throw NullPointerException if the context parameter is null.
