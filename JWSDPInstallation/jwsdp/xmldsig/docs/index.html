Java XML Digital Signature API (JSR 105)


Implementation Version: 1.0 EA

The JSR 105 XML Digital Signature EA implementation, included as part of the JavaTM Web Services Developer Pack 1.4, provides an API that a Java application developer can use to sign and validate XML Signatures. This implementation of JSR 105 is based on Apache's XML-DSig (XML Digital Signature) implementation, which is based on the XML-Signature Syntax and Processing W3C standard.

JSR 105 is undergoing definition under the Java Community Process, and the implementation that is provided in the Java Web Services Developer Pack 1.4 is based on the Proposed Final Draft of the API, which should be fairly stable but is still subject to change until the JSR is finalized.

This distribution includes samples that show how a Java application developer can use the JSR 105 XML Digital Signature API to sign and validate XML Signatures.
